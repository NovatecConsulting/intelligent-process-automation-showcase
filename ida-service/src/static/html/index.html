Intelligent Document Analysis PoC (Rest-API)

Requests

GET /camunda/ida

curl --location --request GET 'localhost:5000/camunda/ida'

Description: To connect to Camunda External Task and execute Intelligent Document Analysis.

POST /ida

curl --location --request POST 'localhost:5000/ida' --form 'file=@"PATH_TO_FILE"'

Description: To call the Intelligent Document Analysis service.The file should be given as form parameter with the name "file".
